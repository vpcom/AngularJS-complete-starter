About this app

This app is an AngularJS starter aimed at helping starting a project or learning.

It is structured in the component-based manner and provides the basic routing with UI-Router. It uses directives for main components and a service factory for delivering the main data. And also, it goes into details with an accute sens of explained good practices.

It also contains HTML5 semantic tags and appropriate CSS3 styling.

Good practices

This app is a lot about good practices, see what is in place:

  • A component based structure to make components self contained in their folder. The routes or states are defined in each module and they also contain their specific CSS.
  • The Immediately Invoked Function Expression (IIFE) syntax in all JavaScript files to separate each script.
  • JavaScript strict mode to help having clean code.
  • Named function to help the debugging.
  • The use of vm (view's model) variable to catch the context of this.
  • The ControllerAs syntax for avoiding using the $scope directly.
  • The dependency injection made separately for allowing non destructive minification and for improved readability.
  • There is a separate app.module.js file to create the main module and inject dependencies.
  • And just in case, ng-cloack is used to avoid the display of uncompiled elements at first display.
